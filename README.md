# @saber2pr/test

> test utils.

```bash
yarn add @saber2pr/test
```

### Example

```ts
import Test from "@saber2pr/test"

Test.watchStackSize(console.log)

while (1) {
  Test()
}
```

or:

```html
<script src="https://saber2pr.top/test/test.min.js"></script>
```

> Author: saber2pr
